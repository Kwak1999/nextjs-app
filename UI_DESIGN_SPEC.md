## 주요 화면 설계

### 1. 네비게이션 바 (공통)

**크기**: 전체 너비 × 64px (높이)

**구조**:
```
┌─────────────────────────────────────────────────────────────┐
│  [로고]              [검색바]              [로그인] [상품등록] │
└─────────────────────────────────────────────────────────────┘
```

**상세 스펙**:
- **배경색**: `--neutral-50` (화이트)
- **테두리**: 하단 `1px solid --neutral-200`
- **높이**: `64px`
- **패딩**: 좌우 `--spacing-6` (모바일), `--spacing-12` (데스크탑)

**요소**:
1. **로고** (좌측)
   - 크기: 120px × 32px
   - 폰트: `--font-bold`, `--text-2xl`
   - 색상: `--primary-600`
   - 아이콘: 🏪 + "중고마켓"

2. **검색바** (중앙, 데스크탑만)
   - 너비: 최대 `600px`
   - 높이: `44px`
   - 배경: `--neutral-100`
   - 테두리: 포커스 시 `2px solid --primary-500`
   - 둥근 모서리: `--radius-full`
   - 아이콘: 돋보기 (좌측 패딩 내)
   - 플레이스홀더: "상품명을 검색하세요"

3. **로그인 버튼** (우측)
   - 크기: `80px × 36px`
   - 배경: 투명
   - 텍스트: `--neutral-700`, `--font-medium`
   - Hover: 배경 `--neutral-100`

4. **상품 등록 버튼** (우측)
   - 크기: `120px × 40px`
   - 배경: `--primary-500`
   - 텍스트: 화이트, `--font-semibold`
   - 둥근 모서리: `--radius-lg`
   - Hover: 배경 `--primary-600`
   - 그림자: `--shadow-md`

**로그인 후**:
- 로그인 버튼 → 프로필 아바타 (40px 원형)
- 클릭 시 드롭다운 메뉴
  - 내 정보
  - 채팅
  - 로그아웃

```
┌─────────────────────────────────────────────────────────────┐
│                        네비게이션 바                           │
├─────────────────────────────────────────────────────────────┤
│  [카테고리 필터]                                              │
├─────────────────────────────────────────────────────────────┤
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                        │
│  │상품1│  │상품2│  │상품3│  │상품4│                        │
│  └─────┘  └─────┘  └─────┘  └─────┘                        │
│  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐                        │

```

#### 카테고리 필터

**크기**: 전체 너비 × 120px

**배경**: `--neutral-50`

**구조**:
```
┌──────────┬──────────┬──────────┬──────────┬──────────┐
│  [아이콘] │  [아이콘] │  [아이콘] │  [아이콘] │  [아이콘] │
│ 디지털기기│   가구    │  생활가전 │   의류    │   기타   │
└──────────┴──────────┴──────────┴──────────┴──────────┘
```

**카테고리 아이템**:
- 크기: `120px × 96px`
- 배경: 선택 전 `transparent`, 선택 후 `--primary-50`
- 테두리: 선택 시 `2px solid --primary-500`
- 둥근 모서리: `--radius-xl`
- 아이콘: `48px × 48px` (중앙 정렬)
- 라벨: `--text-sm`, `--neutral-700`
- 간격: `--spacing-4`

**아이콘 매핑**:
- 디지털기기: 💻
- 가구: 🛋️
- 생활가전: 📺
- 의류: 👕
- 기타: 📦

#### 상품 카드

**크기**: 
- 모바일: 1열, 전체 너비
- 태블릿: 2열, `calc(50% - 8px)`
- 데스크탑: 4열, `calc(25% - 12px)`

**카드 구조**:
```
┌─────────────────────┐
│                     │
│    [상품 이미지]     │  ← 1:1 비율
│                     │
├─────────────────────┤
│ 제목                │
│ ₩2,500,000         │  ← 가격 (bold)
│ 📍 강남구 · 1시간 전 │  ← 위치 & 시간
│               ❤️ 12 │  ← 좋아요 (우측)
└─────────────────────┘
```

**상세 스펙**:
1. **이미지 영역**
   - 비율: `1:1` (정사각형)
   - 배경: `--neutral-200`
   - 둥근 모서리: 상단만 `--radius-xl`
   - 객체 맞춤: `object-cover`
   - Hover: 약간 확대 (scale: 1.05)

2. **좋아요 버튼** (이미지 위 우상단)
   - 크기: `32px × 32px`
   - 배경: `rgba(255, 255, 255, 0.9)`
   - 아이콘: 하트 (빈 하트 → 채워진 하트)
   - 색상: 비활성 `--neutral-400`, 활성 `--rose-500`
   - 둥근 모서리: `--radius-full`
   - 위치: 우상단 8px, 8px

3. **정보 영역**
   - 패딩: `--spacing-4`
   - 배경: 화이트
   - 둥근 모서리: 하단만 `--radius-xl`

4. **제목**
   - 폰트: `--text-base`, `--font-medium`
   - 색상: `--neutral-900`
   - 줄 수: 최대 2줄
   - Overflow: `text-overflow: ellipsis`

5. **가격**
   - 폰트: `--text-xl`, `--font-bold`
   - 색상: `--neutral-900`
   - 여백: 상단 `--spacing-2`

6. **위치 & 시간**
   - 폰트: `--text-sm`, `--font-normal`
   - 색상: `--neutral-500`
   - 아이콘: 📍 (위치), · (구분자)
   - 여백: 상단 `--spacing-2`

7. **좋아요 수**
   - 폰트: `--text-sm`, `--font-normal`
   - 색상: `--neutral-500`
   - 위치: 우하단
   - 아이콘: ❤️

**상호작용**:
- Hover: 그림자 `--shadow-lg`, 약간 상승 (translateY: -4px)
- 클릭: 상품 상세 페이지 이동
